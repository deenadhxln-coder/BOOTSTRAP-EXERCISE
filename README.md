# Frontend Bootcamp — Bootstrap Landing Page

A responsive landing page built as a Bootstrap 5 practice exercise. It covers core Bootstrap layout utilities, components, and responsive design patterns.

Project Structure
project/
├── index.html
├── style.css
└── img/
    ├── undraw_morning-news_h9nz.svg
    ├── undraw_static-assets_di5m.svg
    └── undraw_react_ytu4.svg

How to Run

1. Clone or download the project folder.
2. Place your SVG images inside an `img/` folder (or swap with your own).
3. Open `index.html` directly in a browser — no build step needed.

Built as a frontend practice exercise to learn Bootstrap 5 layout and component usage.
